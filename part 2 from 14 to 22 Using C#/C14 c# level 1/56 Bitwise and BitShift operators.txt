1. معاملات البت (Bitwise Operators)
معاملات البت تُستخدم لتطبيق العمليات على مستوى البتات الفردية للأعداد الصحيحة (int, long, إلخ). هذه العمليات تتضمن AND، OR، XOR، وNOT.

المعامل	الاسم	الوصف
&	AND	يعيد 1 إذا كانت البتات المقابلة في كلا الرقمين 1
`	`	OR
^	XOR	يعيد 1 إذا كانت البتات المقابلة في الرقمين مختلفة
~	NOT	يعكس كل بت في الرقم (أي يُحول 0 إلى 1 والعكس)
أمثلة:
csharp
Copy code
int a = 6;    // 6 = 110 في الثنائي
int b = 3;    // 3 = 011 في الثنائي

int andResult = a & b;  // 110 & 011 = 010 (أي 2)
int orResult = a | b;   // 110 | 011 = 111 (أي 7)
int xorResult = a ^ b;  // 110 ^ 011 = 101 (أي 5)
int notResult = ~a;     // ~110 = 001...1001 (اعتمادًا على حجم المتغير)
2. معاملات إزاحة البت (Bit Shift Operators)
معاملات إزاحة البت تُستخدم لتحريك البتات في العدد لليسار أو لليمين بمقدار محدد. الإزاحة لليمين أو لليسار يمكن أن تكون مفيدة في تحسين العمليات الحسابية مثل الضرب أو القسمة بالقوى الثنائية.

المعامل	الاسم	الوصف
<<	إزاحة لليسار (Left Shift)	يحرك البتات لليسار ويملأ البتات الأقل أهمية بـ 0
>>	إزاحة لليمين (Right Shift)	يحرك البتات لليمين ويملأ البتات الأكثر أهمية بناءً على العلامة (إذا كان موجبًا أو سالبًا)
أمثلة:
csharp
Copy code
int x = 8;  // 8 = 1000 في الثنائي

int leftShift = x << 2;  // 1000 << 2 = 100000 (أي 32)
int rightShift = x >> 2; // 1000 >> 2 = 0010 (أي 2)
الإزاحة لليسار (<<): كل إزاحة بمقدار 1 بت لليسار تضاعف العدد (أي مثل الضرب بـ 2).
الإزاحة لليمين (>>): كل إزاحة بمقدار 1 بت لليمين تقسم العدد على 2.
