    /*
    The clip-path: polygon(50% 0%, 0% 100%, 100% 100%); rule takes the element’s rectangular box and “cuts out” a triangular region defined by three corner-points. Here’s how it works:

        Coordinate System

        The polygon coordinates are given in percentages relative to the element’s own bounding box:

            0% 0% is the top-left corner,

            100% 100% is the bottom-right corner,

            50% 50% would be the exact center, etc.

        Points in Order

            50% 0% → the point halfway along the top edge (the midpoint of the top side).

            0% 100% → the bottom-left corner.

            100% 100% → the bottom-right corner.
    */
    .triangle {
      width: 200px;
      height: 200px;
