1. التحويل الضمني (Implicit Casting)
يحدث التحويل الضمني عندما يتم تحويل نوع البيانات الأصغر إلى نوع بيانات أكبر دون الحاجة إلى تصريح خاص من المبرمج. يتم ذلك بأمان لأنه لن يؤدي إلى فقدان البيانات.

مثال على التحويل الضمني:
csharp
Copy code
int num = 123;
double doubleNum = num;  // التحويل الضمني من int إلى double
Console.WriteLine(doubleNum); // الناتج: 123.0
في هذا المثال، يتم تحويل قيمة num من نوع int إلى double تلقائيًا بدون الحاجة إلى استخدام تعبير تحويل صريح.

أنواع التحويل الضمني المسموح بها:
من int إلى long
من float إلى double
من char إلى int
من أي نوع عدد صحيح أصغر إلى نوع أكبر (مثل byte إلى int)
2. التحويل الصريح (Explicit Casting)
عندما تريد تحويل نوع بيانات أكبر إلى نوع بيانات أصغر، أو تحويل نوعين غير متوافقين، يجب أن تستخدم التحويل الصريح. في هذه الحالة، قد يكون هناك خطر فقدان البيانات، لذا يجب عليك استخدام الصيغة الصريحة لتحذير المترجم بأنك تعلم المخاطر.

مثال على التحويل الصريح:
csharp
Copy code
double doubleNum = 123.45;
int intNum = (int)doubleNum;  // التحويل الصريح من double إلى int
Console.WriteLine(intNum); // الناتج: 123 (تم قطع القيمة العشرية)
في هذا المثال، يتم تحويل double إلى int باستخدام تحويل صريح، مما يعني أن الجزء العشري سيتم تجاهله.

ملاحظات حول التحويل الصريح:
التحويل الصريح قد يؤدي إلى فقدان البيانات (مثل فقدان الجزء العشري عند تحويل double إلى int).
إذا كانت القيمة خارج النطاق المسموح به لنوع البيانات الهدف، قد يؤدي ذلك إلى نتائج غير متوقعة.
3. استخدام Convert للتحويل بين الأنواع
يمكنك أيضًا استخدام طرق التحويل في Convert لتحويل أنواع البيانات المختلفة بطريقة أكثر أمانًا.

مثال:
csharp
Copy code
string strNumber = "123";
int number = Convert.ToInt32(strNumber);
Console.WriteLine(number); // الناتج: 123




C# Type Casting
Type casting is when you assign a value of one data type to another type.

In C#, there are two types of casting:

Implicit Casting (automatically) - converting a smaller type to a larger type size
char -> int -> long -> float -> double

Explicit Casting (manually) - converting a larger type to a smaller size type
double -> float -> long -> int -> char
